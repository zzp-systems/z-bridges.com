zLibrary - A Knowledge Hub by Z-Bridges
Welcome to the zLibrary, the official knowledge hub for the Z-Bridges Company. This project serves as a central repository for strategic frameworks, operational playbooks, core AI resources, and bespoke digital systems.

Overview
This project is a multi-page website designed to provide comprehensive insights into the methodologies and creations of Z-Bridges. It is structured into several key sections:

The Corpus: Contains foundational design systems, including Forgie's Aesthetics Deck and Archie's Architect Deck.

Articles: A collection of articles and insights on topics like Brand Strategy, AI Collaboration, and Business Frameworks.

Digital Originals: A showcase for bespoke systems and case studies developed by Z-Bridges.

Partner Login: A secure portal for partners to access internal documents and assets.

Contact: A commission form for engaging with the Z-Bridges team.

File Structure
The repository is organized as follows:

index.html: The main landing page (inferred, but not provided).

corpus.html: The page for core AI and design systems.

articles.html: The main page for articles and playbooks.

originals.html: The showcase for bespoke projects.

login.html: The partner login portal.

contact.html: The project commission form.

style.css: The central stylesheet, defining the "Futuristic Innovation" theme used across the site.
