Glossary

Archie's Grid-12
A 12-column responsive grid system from the Archie's Architect Deck, used for creating structured, predictable layouts.
Archie's Stack
A layout component from Archie's for vertical rhythm, arranging elements in a single column with consistent spacing.
Archie's CQ (Container Queries)
Refers to components that adapt based on their container's size, rather than the viewport, allowing for more modular design.
Modular
Indicates the aesthetic is designed in self-contained blocks (cards, sections) that can be easily rearranged or reused.
Print-Ready
The aesthetic includes styles and considerations for high-quality printing, such as appropriate color profiles and page layout rules.
