Welcome to the LockBox CK Ledger!

This application helps you manage property lockboxes and keys efficiently. Below is a quick guide to get you started.

How to Use This App

  1. View Dashboard (Here!): Get an overview and access the main ledger.
  2. Access Ledger: Click the "Ledger" button above.
  3. Enter New Information: Use the form at the top of the Ledger view to add new lockbox or key details. Fields marked with * are required.
  4. Save Data: Click "Save Information" to store your entry. The form will reset.
  5. View Overview: The table below displays all property information, populated from a master list and your entries.
  6. Search: Use the search bar to filter properties or units.
  7. Select Multiple: Check boxes next to rows to highlight and group them.
  8. Location Tracking: Your location will be logged upon accessing the app and periodically when it changes.

Functionalities

  • Centralized data entry for lockboxes and keys
  • Persistent storage of your entries
  • Real-time overview of all properties
  • Search and multi-select for data retrieval
  • User activity logging (saves, searches, selections)
  • Location tracking for audit and verification
  • User-specific data visibility

Security & Privacy

Data Storage: All data is stored securely within Google Workspace. Access is restricted based on organizational permissions.

User Identification: Your email is logged with each action for accountability.

Location Tracking:

  • Logs your location when accessing the app
  • Periodically logs location changes while the app is open

Data Usage: Collected data is used solely for property management and accountability.

Your Responsibility: Handle sensitive information responsibly according to company policy.
